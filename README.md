# Edward's Crazy Eights

一个基于 React + Vite + Tailwind CSS 构建的经典“疯狂 8 点”纸牌游戏。

## 游戏特性
- **完整规则**：支持标准 52 张牌，玩家与 AI 各 8 张初始手牌。
- **万能 8 点**：打出数字 8 可自由切换花色。
- **智能 AI**：具有基础策略的 AI 对手。
- **响应式设计**：完美适配手机、平板和桌面端。
- **流畅动画**：使用 Framer Motion 实现丝滑的卡牌交互。

## 本地开发

1. 克隆仓库：
   ```bash
   git clone <你的仓库地址>
   ```
2. 安装依赖：
   ```bash
   npm install
   ```
3. 启动开发服务器：
   ```bash
   npm run dev
   ```

## 部署到 Vercel

本项目已针对 Vercel 进行优化。

1. 将代码推送到 GitHub。
2. 在 [Vercel 控制台](https://vercel.com) 点击 "Add New Project"。
3. 导入你的 GitHub 仓库。
4. Vercel 会自动识别 Vite 配置。
5. 点击 **Deploy** 即可。

## 技术栈
- React 19
- Tailwind CSS 4
- Framer Motion
- Lucide React
